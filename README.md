# GoWiki
A simple wiki page where you can create and view articles. Created using Go and React
![GoWiki logo](images/logo.png)
## Dependencies/Frameworks
### Go
> -  net/http
>  - mongo-driver
>  -  encoding/json
### JavaScript
> - React
> - Axios
